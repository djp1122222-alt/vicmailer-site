# Vicmailer

Email outreach and sales generation system.

## What it does

- Scrape emails from any website
- Send automated email campaigns
- Track your outreach progress
- Simple dashboard to manage everything

## Quick Start

1. Clone this repo
2. Open `index.html` in your browser
3. Or deploy to GitHub Pages

## Files

- `index.html` — Landing page
- `terms.html` — Terms of Service
- `privacy.html` — Privacy Policy
- `about.html` — About page
- `robots.txt` — Crawler instructions
- `.well-known/agi.json` — Agent discovery
